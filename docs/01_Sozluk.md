# 01 · Sözlük

## Vardiya
- **Vardiya modeli:** *tek* (tek blok) veya *çift* (gündüz + gece).
- **Gündüz / Gece:** çift vardiyalı şubelerde sabah ve akşam blokları.
- **Sabahçı / Akşamçı:** o gün gündüz / gece çalışan kişi.
- **Ara vardiya:** belirli saatte (örn. 12:00 / 14:00) gelip 8 saat çalışan kişi; çizelgede saatiyle ve turuncu vurguyla gösterilir.

## Görevler (şubeye özel)
- **Kasa**, **Ocak**, **Garson**, **Mutfak**, **Temizlik** — standart.
- **Aracı:** ocağa yardım eden / izinleri çeviren yardımcı pozisyon.
- **Kahvaltı:** sabah mutfakta ayrı pozisyon (mutfaktan farklı slot).
- **Balıkçı / Dondurmacı / Kasiyer:** diğer şube tipleri.
- **R / C:** garsonun çalıştığı yer — **R**=Restoran, **C**=Cafe.

## Puantaj notasyonu
- **x** = çalıştı
- **i** = haftalık izin
- **R** = rapor (istirahat)
- **Üİ** = ücretsiz izin

## Mesai
- **+saat** = o gün yapılan fazla çalışma (örn. +3 = 3 saat fazla).

## Roller
- **Mesul Müdür / Şube Sorumlusu** — şube yöneticisi.
- **İşletme Müdürü / Merkez** — çok şube denetleyen yetkili.
